# 🎵 Spotify-YouTube-PowerBI-Dashboard
Interactive Power BI Dashboard analyzing Spotify &amp; YouTube music data using Power Query and DAX.


## Project Overview

This Power BI dashboard analyzes Spotify and YouTube music performance using KPIs, interactive charts, and slicers.

## Objectives

- Clean messy dataset
- Handle missing values
- Remove duplicates
- Standardize column names
- Convert data types
- Create interactive dashboard

## Tools Used

- Power BI
- Power Query
- DAX
- Excel

## KPIs

- Total Streams
- Total Views
- Total Likes
- Total Tracks

## Dashboard Features

- Top 10 Artists
- Top 10 Tracks
- Album Type Distribution
- Views vs Streams
- Interactive Filters
- Music Performance Analysis
